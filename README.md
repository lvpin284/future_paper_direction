# future_paper_direction

### 1. TTS 领域：NaturalSpeech 3 (MSR, 2024)

**核心关注点：** 可控合成、音色克隆、属性分解。

* **推荐理由：** 这篇文章提出了​**因子化扩散模型（Factorized Diffusion）**​。它将语音分解为不同的子空间（内容、韵律、音色、细节），这直接回应了你对“可控合成”的需求。
* **研读重点：**
  * 如何通过神经编解码器（Neural Codec）实现属性分离。
  * 其“零样本克隆（Zero-shot Cloning）”的实现机制。
  * 如何利用自然语言或标签在子空间内进行精细化控制。

### 2. 多语言/Omni 领域：SeamlessM4T v2 (Meta, 2023/2024)

**核心关注点：** 跨语言合成、端到端语音翻译（ST）、多语言统一模型。

* **推荐理由：** 这是目前“多语种、多任务”最典型的全能模型（Omni model）。它跳过了中间文本转写，直接实现语音到语音（S2ST）和语音到文本的翻译，支持上百种语言。
* **研读重点：**
  * ​**UnitY2 架构**​：如何实现无缝的跨语言模态转换。
  * ​**多语言对齐**​：在 Top 10 甚至 Top 100 语种中如何保持音色的一致性（Timbre Consistency）。
  * 数据扩增与大规模预训练的策略。

### 3. 多模态 ASR 领域：AV-Hubert (Facebook AI, 2022/2023)

**核心关注点：** 视听双模、口型辅助消歧、环境去噪。

* **推荐理由：** 在多模态 ASR 方向，Audio-Visual Speech Representation (AVSR) 是绕不开的。AV-Hubert 是该领域的基石，利用口型和音频的自监督学习，在极高噪声环境下依然能保持极高的识别率。
* **研读重点：**
  * ​**视听融合策略**​：如何利用视觉特征（口型）辅助音频进行消歧。
  * ​**自监督学习（SSL）**​：在没有海量标注字幕的情况下，如何学习视听一致性。
  * 面对复杂环境（如多人说话、噪声）时的鲁棒性设计。
